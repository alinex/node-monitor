¶ Monitoring contacts 


        		
      

      

        


  

    ¶
    Runtime configuration
  




        		
        




        		
      

      

        


  ¶



Define the server hostname of the machine, which should run all controllers
not specific for a special machine. Keep it empty to make the current machine
the master. controllers which have an runat setting will only run on the
defined hostname.

        		
        
master: pc-alex-linux




        		
      

      

        


  

    ¶
    Default activities
  






They will be used if no other rule defined in controller.

        		
        
rules:
  - # Only work on specific status.
    status: fail



        		
      

      

        


  ¶



Number of minimum attempts before informing.

        		
        
    num: 3



        		
      

      

        


  ¶



Time (in seconds) to wait before informing.

        		
        
    latency: 60



        		
      

      

        


  ¶



Only inform if dependent jobs not failed. This prevents of hundred of
messages if a central system failed.

        		
        
    dependskip: true



        		
      

      

        


  ¶



Timeout (in seconds) without status change before informing again.

        		
        
    redo: '30m'



        		
      

      

        


  ¶



Type of actor to run with it's configuration

        		
        
    email:
      to: plusserver # email address or alias from monitor.contact
      template: fail # template to use defined under monitor.email





        		
      

      

        


  

    ¶
    Contacts
  






With this list you have the possibility to define alias names for contacts
which contains the details or a list of other alias names to use as a group.
After defining it here in a central place you may everywhere use the short
alias names of persons and groups.

        		
        
contacts:




        		
      

      

        


  ¶



Define some groups

        		
        
  plusserver: [ aschi, malbr ]
  centron: [ aschi, malbr, yzhao ]
