¶ Combine values

For multiple dependencies this value defines how the individual sensors are combined to calculate the overall status:

  • max - the one with the highest failure value is used
  • min - the lowest failure value is used
  • average - the average status (arithmetic round) is used
 
combine: max

rule:
  - fail
  - warn
  - ok
