¶ Package: alinex-monitor

This should be an application to make server management easy and fast.

In the moment it is incomplete and not usable. But you may use some of the submodules directly in your own application.

It is one of the modules of the Alinex Universe following the code standards defined there.

¶ Install

The easiest way to use it is to install it with the node package manager:

> npm install alinex-monitor

This will install the package, in your current directory.

¶ Submodules

Currently only the following submodules exists:

  • alinex-monitor-sensor - collection of sensors to check systems and services

In the future there will also be:

  • alinex-monitor-collector - a storage for collected data
  • alinex-monitor-action - an alert system with possible auto repair
  • alinex-monitor-frontend - a web frontend application
  • alinex-monitor-rest - a rest interface to connect to other services
  • alinex-monitor-shell - a command shell for interactive work
